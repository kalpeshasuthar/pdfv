# Features #
  1. GTK+ 3 UI
  1. fitz (MuPDF) rendering engine
  1. continuous page viewer (arrow keys)
  1. zoom in/out (Ctrl + '=' / Ctrl + '-')
  1. jump to a page (g|123g|G)
  1. show/hide bookmarks (Alt + 'b')
